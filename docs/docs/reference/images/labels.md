## Overview

Headjack uses OCI labels to configure container behavior.

## Available Labels

| Label | Description |
|-------|-------------|
| `io.headjack.init` | Command to run as PID 1 |
| `io.headjack.podman.flags` | Podman-specific runtime flags |
| `io.headjack.apple.flags` | Apple Containerization flags |
